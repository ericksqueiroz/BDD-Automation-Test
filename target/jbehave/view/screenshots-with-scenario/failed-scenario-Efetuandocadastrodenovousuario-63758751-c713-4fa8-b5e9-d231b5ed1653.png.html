Como alterar o seu navegador predefinido no Windows 10

}

Este artigo aplica-se apenas ao Windows 10.
Quando atualizar para o Windows 10, poderá inadvertidamente definir o Microsoft Edge como o seu navegador predefinido. Siga os seguintes passos para voltar a definir o Firefox como o seu navegador predefinido.

  1. Clique no botão do menu e escolha Opções.
  2. No painel Geral clique em Tornar pré-definido.
  3. A janela de Programas predefinidos será apresentada.
  4. Na janela de Programas predefinidos, selecione Firefox da lista de programas à esquerda e clique em Definir este programa como predefinido. Depois clique OK para fechar a janela.
  5. Feche a página about:preferences. Quaisquer alterações feitas serão guardadas automaticamente.
  1. Clique no botão do menu e escolha Opções.
  2. No painel Geral clique em Tornar predefinido.
  3. A aplicação de Definições do Windows irá apresentar o ecrã de Escolher aplicações predefinidas.
  4. Deslize para baixo e clique na entrada sob Browser. Neste caso, o ícone irá indicar o Microsoft Edge ou Escolher uma predefinição.
  5. No ecrã de Escolher uma aplicação, clique em Firefox para o definir como o navegador predefinido.
  6. O Firefox está agora definido como o seu navegador predefinido. Feche a janela para guardar as suas alterações.
